# A Great Idea With A Sad Ending
The original idea here was to create a Dota 2 mod where the bot team had extra help: things like healer minions, builders creating golems, and other special events to make bot games challenging.

It works locally (or at least it did. Can't quite remember what state I left it in), but ultimately it was killed because Dota has an issue where if you play a mod through their arcade system (required for multiplayer), bots drop down to a super-stupid AI. The only way to have bot AI in a mod is for the mod to specifically be for bot AI and only that. 

So for now, this is retired. If Valve ever changes that limitation/bug, I will pick this back up.
